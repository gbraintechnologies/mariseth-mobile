# MARISETH FARMS MOBILE APPLICATION

## Get started

- Go to the project's root folder and run

```bash
npm install
```

- If you are using Xcode 12.5 or higher got to /ios and execute

```bash
pod install
```

- Run

```bash
npm start
```

```bash
npm run ios
```

OR

```bash
npm start
```

```bash
npm  run android
```

to start the application!
